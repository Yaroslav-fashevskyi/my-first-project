# Contributing

Дякуємо за інтерес до цього проєкту.

## Як зробити внесок
1. Створіть окрему гілку від `main`
2. Внесіть свої зміни
3. Перевірте, що GitHub Actions проходить успішно
4. Зробіть зрозумілий commit
5. Створіть Pull Request

## Рекомендації
- Робіть невеликі логічні зміни
- Використовуйте зрозумілі назви гілок
- Пишіть інформативні commit messages
- Не додавайте зламаний код у `main`
