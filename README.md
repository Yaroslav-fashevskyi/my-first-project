# My First Project

![CI](https://github.com/Yaroslav-fashevskyi/my-first-project/actions/workflows/ci.yml/badge.svg)

## Опис проєкту
Це навчальний проєкт для вивчення Git, GitHub, гілок, Pull Request, Issues, Projects, GitHub Actions і GitHub Pages.

## Можливості
- базова HTML-сторінка
- підключені CSS-стилі
- footer
- секція контактів
- CI через GitHub Actions

## Структура проєкту
- `index.html` — основна сторінка
- `styles.css` — стилі
- `.github/workflows/ci.yml` — CI workflow
- `README.md` — документація
- `CONTRIBUTING.md` — правила внеску

## Встановлення
1. Відкрийте репозиторій на GitHub
2. Склонуйте його або скачайте ZIP
3. Відкрийте папку з проєктом

## Запуск
Відкрийте файл `index.html` у браузері.

## Приклад використання
Проєкт можна використати як шаблон простої статичної сторінки для навчання роботі з GitHub.

## Як зробити внесок
1. Створіть нову гілку
2. Внесіть зміни
3. Зробіть commit
4. Створіть Pull Request
